Analysis Summary

During Milestone One, I performed a thorough review of my Contact Service application across three areas: software design, algorithms & data structures, and databases. Key findings:

  • Strong class structure and unit tests.
  • Repeated validation logic and limited error handling.
  • Linear searches in ArrayLists were inefficient.
  • No persistent database storage was implemented.

Planned Enhancements

  • Refactor input validation and access control for maintainability.
  • Improve performance with HashMap-based data structures.
  • Add a relational database for secure, persistent data storage.
  • Strengthen unit tests, including negative and exception-based tests.
  • Document enhancements clearly in the ePortfolio.
