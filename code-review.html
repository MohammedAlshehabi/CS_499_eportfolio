Code Review Overview

During Milestone One, I conducted a thorough review of my Contact Service application against professional software engineering standards. This self-directed code review identified strengths to build upon and areas requiring enhancement across three critical dimensions:

Enhancement Strategy

Planned Improvements by Artifact

1. Software Design & Engineering Enhancement

  • Refactor validation: Extract repeated validation logic into reusable methods
  • Improve encapsulation: Enhance access control and defensive programming patterns
  • Exception handling: Implement custom exceptions with meaningful error messages
  • Expand testing: Add negative tests and edge case scenarios
  • Documentation: Add comprehensive javadoc and inline comments

2. Algorithms & Data Structures Enhancement

  • Replace ArrayList: Implement LinkedHashMap for O(1) lookups
  • Optimize operations: Reduce search complexity from O(n) to O(1)
  • Eliminate duplication: Refactor repeated search logic
  • Performance analysis: Document improvements with complexity analysis
  • Test performance: Benchmark original vs. enhanced implementation

3. Databases Enhancement

  • Database integration: Add SQLite for persistent data storage
  • CRUD operations: Implement full Create, Read, Update, Delete functionality
  • Security: Use prepared statements to prevent SQL injection
  • Performance: Add database indexing on frequently queried columns
  • Integration testing: Create database-backed unit tests
